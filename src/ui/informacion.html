Expresiones regulares implementados para la validación de los datos
Nombre:
/^([A-ZÁÉÍÓÚ]{1}[a-zñáéíóú]+[\s]*)+$/
Valída nombres que inicien con cualquer letra o vacal, con acento. restricción: Tiene que empezar con mayúscula.
Dirección:
/^[a-zA-Z0-9\s\.\-]+\,[A-Za-zñáéíóú\s]+\,[A-Za-zñáéíóú\s]+$/
Valída direcciones en formato: calle+ciudad+provincia, utilizando números,letras,espacio,coma,y guion.
Constraseña:
/^.{8,20}$/
valida password con cualquier caracter mayor a 8 unidades.
Correo Electrónico:
/^[a-zA-Z0-9_.+-]+@[a-z0-9-]+\.[a-z0-9-.]+$/
valída correo con cualquier caracter, posterior a la @ todos son minuscula.
Género:
/^([f|m|F|M]{1}[a|e]{1}[c-u]+[n|l]{1}[ino])$/
valída unicamente las opciones de femenino ó masculino.
Dpi cui:
(1) /^[0-9]{4}[0-9]{5}[0-2]{1}[0-9]{1}[0-2]{1}[0-9]{1}$/
(2) /^([0-9]{4})\-([0-9]{5})\-([0-2]{1}[0-9]{1}[0-2]{1}[0-9]{1})$/
Valída numeros de dpi de 13 digitos, con la restricción de los ultimos 4, que tiene que cumplir con el rango de departamento y municipio.
(1)sin '-' (2)con'-'
Nit código tributario:
/^([0-9]{6,7})\-([0-9]{1})$/
valída unicamente numeros de 6 a 8 digitos, antes del ultimo digito debe ir el guión.
Código Postal:
/^[0-2]{1}[0-9]{1}[0-2]{1}[0-9]{1}$/
valida si es correspondiente al rango de municipio y departemento.
Número teléfono:
(1) /^[5,0,2]{3}[4|5|7]{1}([\d]{7})$/
(2) /^[4|5|7]{1}([\d]{7})$/
(1)Valida el número de teléfono con el código del país 502.
(2)Valida el número de teléfono sin el código del país.
Fecha:
(1) /^(0[1-9]|1\d|2\d|3[01])\/(0[1-9]|1[0-2])\/(19|20)\d{2}$/
(1)valida formato de fecha con diagonales.|20/01/2020|

Formulario de datos
Aplicación de escritorio para validar datos por medio de expresiones regulares.

Detalles de la aplicación.

  • Lengueaje de Programación: Javascript
  • Herramientas de diseño: html5 , css3 'boostrap4'., Chorminum
  • Frameworks: electron js, node js
  • Descripción:
    • La apliación multiplataforma ('windows,MACos,linux'), de arquitectura de 64x.bits, para escritorio. Implementa expresiones regulares para validar datos: Nombre, Dirección, Género, rango de fechas, Correo electronico y contraseña.
    En contexto del pais de Guatemala: número de teléfono, DPI 'documento personal de identificación', Nit 'Número de identificación tributario', y Código postal.
    Valída cada uno de los datos ingresados, mostrando un indicardor de exito ó de error del campo.
    Lista el registro de datos validados, y los almacena en memoria dinámica con la opcion de borrar.
